# 基于E53_SC2扩展的BearPI-HM_Nano倾角测量器开发

利用E53_SC2的三轴传感器，测量相对于初始状态的Pitch和Roll，并通过F1按钮保存倾角数值便于NFC设备读取，通过F2按钮将倾角数值上传华为云平台。

# 特点

Reset键可以重置初始状态的Pitch和Roll。

往左倾时，E53_SC2扩展板的蓝灯会亮，此时测量的是Pitch角（为正数）；往右倾时，E53_SC2扩展板的红灯会亮，此时测量的是Pitch角（为正数）；往前倾时，BearPi的Led灯会亮，此时测量的是Roll角（为负数）。当水平放置时，灯全灭（除了power灯）。

F1键可以将测量的角度数值保存，用具有NFC读取功能的设备（如NFC手机）读取，可以得到Pitch和Roll，实现现场读取的需要。

F2键可以将测量的角度上传至华为云平台，实现远程读取的需要。

未处理传感器内部误差和传感器的位置可能不水平，角度测量存在一定误差。
